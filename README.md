# ABDM-HIU-SDK
SDK for ABDM HIU


Installation
------------

    pip install abdm-hiu-sdk
------------
The current version of the SDK is 0.1.0

# Getting Started



# Authenticate With ABDM
``` python
from auth import ABDM_HIU_PATH

auth = ABDM_HIU_PATH(
    clientId='YOUR_ABDM_CLIENT_ID',
    clientSecret='YOUR_ABDM_CLIENT_SECRET'
)

```
